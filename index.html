Initial setup on a laptop with Windows preinstalled
  • 0. power on the laptop and connect the charger if battery is less than 50% then set a battery charge limit of 80%
  • 1. check the hardware (Screen, ports, battery health, keyboard, touchpad, thermals)
  • 2. install (if there any) the newest SSD firmware (EX: using Samsung Magician for Samsung SSDs)
  • 3. save the activation key (if present) in a safe place using ShowKeyPlus from the Microsoft Store
  • 4. reinstall a clean version of windows 11 and configure BitLocker
  • 5. install Windows updates and the software on the manufacturer site including drivers and restart acordingly (exlude BIOS/UEFI for now)
  • 6. remove the bloatware (The preinstalled uneeded software)
  • 7. unlink OneDrive
  • 8. turn off the uneeded auto launch apps in settings then restart the laptop
  • 9. disable data collection in the privacy settings (Also use OOSU10, link is provided under Security & Privacy in the Windows section)
  • 10. let it charge
  • 12. while on ac power and the battery having at least 50% charge install the latest BIOS/UEFI update
  • 13. check the screen's resolution and the refresh rate
  • 14. enable showing file extentions in the file manager
  • 15. personalize and turn on dark mode change wallpaper and customize the desktop&taskbar (no need for heavy customization rn)
  • 16. enable memory integrity in core isolation settings if not enabled
  • 17. enable/install windows sandbox
  • 18. disable scripts from running in the group policy editor or using PolicyPlus (listed below)
  • 19. install the needed software
